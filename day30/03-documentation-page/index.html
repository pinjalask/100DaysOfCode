Introduction

Python is a high-level, interpreted programming language known for its simplicity and versatility. Developed by Guido van Rossum and first released in 1991, Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming. It is widely used for web development, data analysis, artificial intelligence, and more.

Installation

  1. Download Python:

    • Visit python.org.
    • Download the latest version for your operating system.

  2. Install Python:

    • Run the installer and ensure you check the "Add Python to PATH" option.
    • Follow the setup instructions to complete the installation.

  3. Verify Installation:

    • Open a terminal or command prompt and type:
      • python --version
Basic Syntax
Printing Output

To display output on the screen, use the print() function:

print("Hello, World!")
Variables

Python allows dynamic typing. You can assign values to variables without declaring their types:

x = 5
name = "Python"
Data Types

Python includes several built-in data types:

  • Numbers: int, float
  • Strings:Enclosed in single or double quotes ("Hello" or 'World')
  • Lists:Ordered, mutable collections ([1, 2, 3])
  • DictionariesKey-value pairs ({"key": "value"})
Control Flow
Conditional Statements

Conditional statements allow you to execute code based on conditions:

if x > 0:
  print("Positive")
elif x == 0:
  print("Zero")
else:   print("Negative")
Loops
For Loop

Use for loops to iterate over a sequence:

for i in range(5):
  print(i)
While Loop

Use while loops to execute a block of code as long as a condition is true:

while x > 0:
  print(x)
  x -= 1
Functions
Defining a Function

To read from a file:

def greet(name):
   return f"Hello, {name}!"
Calling a Function

Invoke the function by passing required arguments:

print(greet("World"))
Modules
Importing Modules

Modules allow you to use pre-written code:

import math
print(math.sqrt(16))
Creating a Module

Save the following code in a file named my_module.py:

def add(a, b):
  return a + b

Import and use it in another script:

from my_module import add
print(add(3, 5))
File Handling
Reading a File

To read from a file:

with open("file.txt", "r") as file:
   content = file.read()
   print(content)
Writing to a File

To write data to a file:

with open("file.txt", "w") as file:
   file.write("Hello, Python!")
Conclusion

Python's beginner-friendly syntax and vast ecosystem make it one of the most popular programming languages. Start exploring its basic features, and as you grow, delve into advanced libraries and frameworks to unlock its full potential!
